# LimbusAuto-Test

《边狱巴士》（Limbus Company）的 Windows 桌面自动化脚本试验版。  
通过图像识别与模拟点击/键盘操作，自动完成日常任务、战斗等重复性操作。
现在只是开始版本，后续进行慢慢完善。作为我个人学习，和分享使用。

## ✨ 当前功能

- 自动领取每日奖励
- 自动换饼
- 自动进入经验本/纺锤本
- 可配置的截图识别与重试机制

## 🛠️ 技术栈

- Python 3.11
- OpenCV（模板匹配）
- PyAutoGUI（鼠标键盘模拟）
- PyWin32（窗口管理）
